pub fn max_past_epochs(self, max_past_epochs: usize) -> Self

Sets the max_past_epochs property of the MlsGroupConfig. This allows application messages from previous epochs to be decrypted.

WARNING

This feature enables the storage of message secrets from past epochs. It is a trade-off between functionality and forward secrecy and should only be enabled if the Delivery Service cannot guarantee that application messages will be sent in the same epoch in which they were generated. The number for max_epochs should be as low as possible.
