TLS Codec

This crate implements the TLS codec as defined in RFC 8446 as well as some extensions required by MLS.

With the feature derive TlsSerialize and TlsDeserialize can be derived.

This crate provides the following data structures that implement TLS serialization

  • u8, u16, u32, u64
  • TlsVecU8, TlsVecU16, TlsVecU32
  • [u8; 2], [u8; 4], [u8; 8], [u8; 16], [u8; 32], [u8; 64]
  • If the hpke feature is enabled, the TLS codec is implemented for HPKEPublicKeys from hpke-rs.
